# React Weather App

A modern and responsive Weather Application built using React, Axios, and Tailwind CSS.
The app allows users to search weather details by city name and fetches real-time data from WeatherAPI. 
It includes input validation, loading states, error handling, and dynamic background changes based on weather conditions.

# Key Highlights
- Real-time weather data using API integration
- City-based search functionality
- Dynamic UI updates with React state
- Loading and error handling
- Clean and responsive design using Tailwind CSS

# Technologies Used
- React (Vite)
- Axios
- Tailwind CSS
- WeatherAPI
